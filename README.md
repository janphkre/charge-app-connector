# charge-app-connector
Use ADB to access data of a connected EV car.

How to use:

Have an old phone setup with the car app.
Necessary settings:
- Keep screen on while charging.
- ADB USB Debugging
- Coordinates for the pull to refresh & climate button can be found out by enabling the developer setting to show information about touch events.
- Have the carapp open on the phone.
Consider having the battery removed from the phone or put it into a fire-protection box.
